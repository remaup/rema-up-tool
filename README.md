# 리마업 병원 블로그 진단 리포트 생성기

콜드 아웃리치용 병원 블로그 진단 PDF를 자동으로 만들어주는 웹 도구입니다.
키워드와 블로그 주소를 입력하면 네이버 검색 API로 현재 순위를 자동 조회해
7페이지 분량의 진단 리포트를 생성합니다.

## 주요 기능
- 병원 정보 입력 → 7페이지 진단 PDF 자동 생성
- 네이버 블로그 검색 API로 키워드 순위 자동 조회
- 미리보기 실시간 갱신 → 브라우저 인쇄로 PDF 저장

## 파일 구조
```
.
├── index.html             메인 페이지 (UI + 미리보기)
├── api/
│   └── check-rank.js      서버리스 함수 (네이버 API 호출)
├── package.json
├── .gitignore
└── 배포_가이드.md           ← 처음 시작은 여기부터
```

## 빠른 시작
1. `배포_가이드.md` 파일을 열어 처음부터 따라 하세요.
2. 네이버 API 발급 → GitHub 업로드 → Vercel 배포 순으로 진행됩니다.
3. 30분~1시간 내 완료됩니다.

## 환경변수
배포 시 Vercel에 다음 환경변수를 등록해야 합니다.
- `NAVER_CLIENT_ID`
- `NAVER_CLIENT_SECRET`
